imagemin CLI is a command-line tool that compresses and optimizes images with minimal effort. It wraps the [imagemin](https://github.com/imagemin/imagemin) library and exposes it as a fast, scriptable CLI — ideal for build pipelines, CI workflows, and one-off batch compressions.

## Supported formats

imagemin CLI ships with optional default plugins that cover the most common image formats:

| Format | Plugin | Description |
|--------|--------|-------------|
| PNG | `imagemin-optipng` | Lossless PNG compression |
| JPEG | `imagemin-jpegtran` | Lossless JPEG optimization |
| GIF | `imagemin-gifsicle` | GIF compression and optimization |
| SVG | `imagemin-svgo` | SVG optimization via SVGO |
| WebP | `imagemin-webp` (custom) | WebP encoding with quality control |

## Quick example

```bash
# Install globally
npm install --global imagemin-cli

# Minify all images in a directory
imagemin images/* --out-dir=build

# Minify a single file
imagemin logo.png > logo-optimized.png

# Use with custom plugin options
imagemin photo.png --plugin.pngquant.quality=0.6 --plugin.pngquant.quality=0.8 > photo-small.png
```
